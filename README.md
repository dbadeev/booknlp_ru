# BookNLP-ru Core

Базовый морфосинтаксический пайплайн для русской художественной литературы.

## Локный запуск (macOS)

1. `python3.11 -m venv .venv && source .venv/bin/activate`
2. `pip install -r requirements.txt`
3. `python -c "from natasha import Segmenter; print('✅ Natasha OK')"`

## PyCharm
- Python Interpreter: `.venv/bin/python`
